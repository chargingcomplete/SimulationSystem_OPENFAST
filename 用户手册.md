# OpenFAST 批量化仿真系统 - 用户手册

## 系统概述
本系统是基于 Streamlit 开发的 OpenFAST 风电仿真批量处理系统，支持单个仿真配置、批量案例管理、结果可视化和智能语言指导。

**主界面包含5个功能板块：**
1. 🔧 单个仿真
2. 🔢 批量化仿真
3. 📊 可视化分析
4. 🎬 VTK 3D可视化
5. 💬 语言指导

---

## 【侧边栏配置】

侧边栏位于页面左侧，提供系统配置和AI助手功能。

### 🔧 OpenFAST 路径设置

**功能：** 指定 OpenFAST 可执行文件的路径

**操作方式：**
1. 使用默认路径（系统预设）
2. 勾选"自定义"后选择自己的 OpenFAST 安装目录

**步骤：**
1. 勾选"自定义"复选框
2. 点击"浏览"按钮选择 OpenFAST 所在目录
3. 系统自动查找 `openfast*.exe` 文件
4. 或手动输入完整的可执行文件路径

### 📂 结果保存目录

**功能：** 设置仿真案例和结果文件的保存位置

**默认目录：** `v8/results`

**操作方式：**
1. 勾选"自定义"复选框
2. 点击"浏览"选择结果保存目录
3. 或手动输入完整路径

**注意：** 该目录下需要有 `- 5MW_Baseline` 基础文件夹

### 📋 模板 FST 信息

**功能：** 显示当前使用的默认模板文件

**说明：** 系统使用 `5MW_OC4Semi_WSt_WavesWN.fst` 作为默认模板

### 💬 AI智能询问助手

**功能：** 通过自然语言询问系统使用问题

**使用方法：**
1. 点击"打开助手"按钮展开面板
2. 在输入框中输入问题（最多100字）
3. 点击"提问"获取回答
4. 查看对话历史记录

**助手能力：**
- 回答系统前端使用问题
- 基于用户手册提供精准解答
- 支持操作步骤指导
- 参数含义说明

**使用限制：**
- 不回答API、后端、部署等技术问题
- 不回答与系统无关的问题
- 输入不超过100字，回答不超过3句话

**常见问题示例：**
- "如何修改仿真时间？"
- "如何关闭VTK输出？"
- "批量仿真怎么导入案例？"
- "结果文件保存在哪里？"

## 【板块1: 单个仿真】

**功能：** 配置并运行单个 OpenFAST 仿真案例

### 操作步骤：
1. **选择 FST 文件**
   - 上传自己的 FST 文件，或勾选"使用默认模板"
   - 如果上传文件，需要指定依赖文件目录（包含 .dat 等文件）

2. **配置仿真参数**
   - 仿真控制：TMax（总仿真时间）、DT（时间步长）等
   - 功能开关：控制各个计算模块的启用状态
   - 环境条件：重力、空气密度、水密度等
   - 输出设置：控制结果输出格式和频率
   - 线性化：线性化分析设置
   - 可视化：VTK 输出设置

3. **案例命名** - 输入案例名称

4. **运行仿真**
   - 点击"更新参数并创建案例"创建案例
   - 点击"运行仿真"开始计算

## 【板块2: 批量化仿真】

**功能：** 批量导入和管理多个仿真案例，一次性运行多个仿真

### 操作步骤：
1. **导入案例文件夹（3种方式）**

   **方式1 - 浏览父文件夹：**
   - 点击"浏览"选择包含多个案例的父目录
   - 勾选要导入的案例文件夹
   - 点击"批量导入"

   **方式2 - 输入路径：**
   - 每行输入一个案例文件夹路径
   - 点击"批量导入"

   **方式3 - 快速添加：**
   - 点击"选择文件夹"逐个添加

2. **编辑案例参数**
   - 点击案例旁的"编辑"按钮
   - 在展开的界面中修改参数
   - 点击"保存参数"

3. **批量创建和运行**
   - 输入案例名称前缀
   - 点击"保存所有配置并创建案例"
   - 点击"运行批量仿真"

## 【板块3: 可视化分析】

**功能：** 查看和分析仿真结果数据（.out/.outb 文件）

### 操作步骤：
1. **选择结果文件夹** - 点击"浏览"选择包含 .out 或 .outb 文件的文件夹

2. **查看数据预览** - 系统自动显示前10行数据

3. **绘制图表**
   - 选择要绘制的指标（支持多选）
   - 选择图表类型（折线图/散点图/面积图）
   - 点击"生成图表"

### 常用指标说明：
| 指标 | 说明 |
|------|------|
| Time | 时间 |
| GenSpeed | 发电机转速 |
| GenPwr | 发电机功率 |
| BldPitch1/2/3 | 叶片变桨角 |
| PtfmSurge/Sway/Heave | 平台纵荡/横荡/垂荡 |
| PtfmPitch/Roll/Yaw | 平台纵摇/横摇/艏摇 |
| FAIRTEN1-10 | 锚链张力 |

## 【板块4: VTK 3D可视化】

**功能：** 3D 可视化 VTK 格式的仿真结果

### 操作步骤：
1. **选择 VTK 文件夹** - 点击"浏览"选择包含 VTK 文件的文件夹

2. **选择显示模式**
   - 动画模式：播放连续动画
   - 单帧模式：查看单个时间步

3. **动画模式操作**
   - 点击"播放"开始动画
   - 调整播放速度
   - 选择是否显示网格边框/坐标轴

4. **单帧模式操作**
   - 拖动滑块选择时间步
   - 使用"快速定位"切换视图角度
   - 鼠标左键拖动旋转，滚轮缩放

---

## 【板块5: 语言指导】

**功能：** 通过自然语言描述需求，自动生成批量仿真案例

### 操作步骤：
1. **选择 FST 模板文件** - 点击"浏览文件"选择模板

2. **输入仿真需求**
   - 示例："创建TMax分别为1,2,3的三个案例"
   - 示例："生成总仿真时间为10、20、30的案例，WrVTK都设为0"

3. **智能生成案例** - 点击"智能生成案例"

4. **确认和修改参数** - 查看并编辑每个案例的参数

5. **创建并运行** - 点击"创建所有案例"，然后"运行批量仿真"

## 【常见问题】

**Q: 如何修改仿真时间？**
A: 在"单个仿真"或"批量仿真"的编辑界面，找到"仿真控制"标签，修改 TMax 参数。

**Q: 如何关闭 VTK 输出？**
A: 在"可视化设置"中，将 WrVTK 设置为 0。

**Q: 结果文件保存在哪里？**
A: 默认保存在 results 目录，可在主界面设置自定义结果目录。

**Q: 如何查看锚链张力？**
A: 在"可视化分析"中选择 FAIRTEN1-10 指标进行绘图。

## 【参数名称对照表】

| 中文 | 参数名 | 说明 |
|------|--------|------|
| 总仿真时间 | TMax | 仿真总时长（秒） |
| 时间步长 | DT | 推荐的模块时间步长（秒） |
| VTK输出 | WrVTK | 0=无, 1=初始化, 2=动画, 3=模态 |
| VTK类型 | VTK_type | 1=曲面, 2=线框, 3=全部 |
| VTK帧率 | VTK_fps | 每秒帧数 |
| 重力加速度 | Gravity | m/s² |
| 空气密度 | AirDens | kg/m³ |
| 水密度 | WtrDens | kg/m³ |
| 水深 | WtrDpth | 米 |
| 结构动力学 | CompElast | 1, 2, 3 |
| 气动载荷 | CompAero | 0=无, 1, 2, 3 |
| 水动力载荷 | CompHydro | 0=无, 1 |
| 系泊系统 | CompMooring | 0=无, 1, 2, 3, 4 |
